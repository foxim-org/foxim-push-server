## FOXIM PUSH SERVER

### How to install

```bash
git clone https://github.com/foxim-org/foxim-push-server
cd
npm i
````

### How to dev

```bash
npm run dev
```

### 文档

> 开发模式可以访问

http://$HOST:$PORT/docs

### ENV

```
PORT=
MQTT_MONGO_URL=
PERSISTENCE_TTL=
JWT_SECRET=
MQTT_SECRET_CLIENT_ID=
REDIS_HOST=
REDIS_PORT=
REDIS_USERNAME=
REDIS_PASSWORD=
```
